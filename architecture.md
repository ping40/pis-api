
目录/文件 | 对应功能
config	| 配置模块
core	| 核心模块（申明过滤器、管道、拦截器、守卫、中间件、全局模块）
feature	| 特性模块（主要业务模块）
shared	| 共享模块（共享mongodb、redis封装服务、通用服务）
tools	| 工具（提供一些小工具函数）
